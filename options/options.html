Custom Shortcut Settings

About Website Shortcuts

This extension allows you to create shortcuts to your favorite websites that can be accessed via:

  • The floating shortcut widget on any webpage
  • Optional keyboard shortcuts (requires additional configuration)

You can set up to 10 keyboard shortcuts for your most frequently used websites.

Default Shortcuts:

  • Ctrl+Shift+U - Open this settings page
  • Alt+Shift+S - Toggle shortcuts widget

Custom URL Keyboard Shortcuts

Hot to add custom shortcuts to directly open specific websites with your chosen keyboard shortcuts.

Important Instructions:

  1. Add your custom URL shortcuts below
  2. After adding a shortcut, go to chrome://extensions/shortcuts
  3. Find this extension and set the keyboard combination for your shortcuts
  4. Return to this page to manage your shortcuts

Chrome extensions require keyboard shortcuts to be configured through Chrome's settings.
